<h2>How to use</h2>
<p>

```

- Clone o repositório:
$ git clone https://github.com/RegisSalesRA/qrCode_grocery_flutter.git

- Entre no diretório:
$ cd qrCode_grocery_flutter

- Instale as dependências:
$ flutter pub get / flutter packages get

- Execute:
$ flutter run

```

</p>
